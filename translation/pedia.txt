# Slimes
t.pink_slime = 粉紅史萊姆
m.intro.pink_slime = 牠們的粉紅色調已被科學證明是愉快的顏色。
m.diet.pink_slime = 水果、蔬菜、肉類
m.favorite.pink_slime = （無）
m.slimeology.pink_slime = 粉紅史萊姆是遠遠星球上最常見的史萊姆。牠們開朗、溫馴，\
	是最容易飼養的史萊姆。粉紅史萊姆會吃下放在牠們面前的任何東西，不過沒有特別愛吃的食物，\
	還有牠們的結晶是所有史萊姆中價值最低的。
m.risks.pink_slime = 不必太過擔心粉紅史萊姆。牠們柔軟、濕黏，喜歡環繞在牧場主身旁，\
	而且野外隨處可見。但正因如此，牧場主到處能發現會導致混種產生的粉紅結晶，\
	然後……
m.plortonomics.pink_slime = 粉紅結晶是一種優異的「多用途物質」，可用來生產任何東西，\
	從食物到家用清潔品。在地球的各個角落，人們用粉紅結晶製品來製作漢堡、調味咖啡\
	或清潔地板。由此可見它的多用途！

m.slimeology.rock_slime = 岩石史萊姆的名字來源於牠們黏滑的身上所突起的尖刺石狀頭冠。\
	但牠們對礦物的喜愛不止於牠們的外表：岩石史萊姆是徹底的素食主義者，\

m.risks.rock_slime = 岩石史萊姆的危險相當顯而易見：牠們的石狀頭冠鋒利到能穿透牧場主的靴子。\
	但單單注意你的腳下還不夠，因為這些岩石老愛滾來滾去。\
	當你看到岩石史萊姆正在加速旋轉時，最好是避開牠。
m.plortonomics.rock_slime = 岩石結晶被看作是生產「藍金屬」的重要成分。\
	藍金屬是一種驚人的合金，比鈦金屬還硬，卻比塑膠還輕。因其驚人的強度，\
	它通常是製造光速星艦的船身、人造四肢和全息手機保護殼\
	的重要成分。
